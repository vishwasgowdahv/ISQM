# Exercise 2

### TODOS

Create Todo ( POST )

```bash
https://api.freeapi.app/api/v1/todos
```

Read Todo ( GET )

```bash
https://api.freeapi.app/api/v1/todos
```

Update Todo( PATCH )

```bash
https://api.freeapi.app/api/v1/todos/{id}
```

Delete Todo ( DELETE )

```bash
https://api.freeapi.app/api/v1/todos/{id}
```
